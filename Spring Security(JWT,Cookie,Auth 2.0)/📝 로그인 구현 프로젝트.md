# JWT 기반 인증 및 인가 전체 프로세스

1. 최초 인증(로그인) : 사용자가 ID/PW 로 인증을 진행하고 API 접근을 위한 **토큰들을 발급받는 과정**
2. API 요청 및 인가 : 발급받은 AccessToken 을 사용하여 접근
3. AccessToken 만료 및 재발급 : RefreshToken 을 사용하여 새로운 AccessToken 재발급
