## 문제 121

>[!.questions]- 정답
>정답 : A

> - RDS에서는 한 번 암호화 없이 생성된 DB 인스턴스를 나중에 직접 수정해서 암호화를 활성할 수 없음. 암호화는 인스턴스를 처음 생성할 때만 설정 가능

> 1. 기존 DB의 스냅샷을 생성
> 2. 스냅샷을 복사하면서, 복사 옵션에서 암호화 활성화 선택
> 3. 암호화된 새 스냅샷을 사용하여 새 DB 인스턴스를 복원

> B : RDS 암호화는 AWS가 관리하는 방식임. 사용자가 직접 EBS 볼륨을 만들어서 스냅샷을 수동으로 옮기는 방식이 아님. 인스턴스에서 암호화 활성화 기능은 존재하지 않음
> C : 스냅샷을 복원할 때는 항상 새로운 DB 인스턴스가 생성됨. 기존 인스턴스 위에 덮어쓰는 방식의 복원은 불가능
> D : S3에 스냅샷 저장은 백업, 분석용으로는 좋지만, 현재 실행 중인 DB 인스턴스 자체를 암호화하는 못함

- **B.** Amazon S3 버킷을 생성합니다. SSL 인증서를 S3 버킷으로 마이그레이션합니다. EC2 인스턴스가 SSL 종료를 위해 버킷을 참조하도록 구성합니다.
    
- **C.** 또 다른 EC2 인스턴스를 프록시 서버로 생성합니다. SSL 인증서를 새 인스턴스로 마이그레이션하고 기존 EC2 인스턴스로 연결을 지시하도록 구성합니다.
    
- **D.** SSL 인증서를 AWS Certificate Manager(ACM)로 가져옵니다. ACM의 SSL 인증서를 사용하는 HTTPS 리스너를 갖춘 Application Load Balancer를 생성합니다.

>[!.questions]- 정답
> 정답 : D
> EC2가 수학 연산을 ALB에게 대신 시켜서 연산을 인스턴스에 담당하지 않도록 한다.
> 자체 인증서를 ACM으로 가져오면, ALB에게 해당 인증서를 선택하여 통신할 수 있다.

>[!.questions]- 문항별 풀이
> A : ACM에서 생성한 인증서는 프라이빗 키를 추출 못해서 EC2 인스턴스 내부에 직접 설치 못함. ACM은 ALB,CloudFront, API Gateway 같은 서비스에서만 사용 가능


## 문제 127

미디어 회사가 시스템을 AWS 클라우드로 마이그레이션할 가능성을 평가하고 있습니다. 이 회사는 비디오 처리를 위해 가능한 최대 I/O 성능을 제공하는 최소 10TB의 스토리지, 미디어 콘텐츠 저장을 위한 300TB의 내구성이 매우 뛰어난 스토리지, 그리고 더 이상 사용되지 않는 아카이브 미디어에 대한 요구 사항을 충족하기 위한 900TB의 스토리지가 필요합니다. 솔루션 아키텍트는 이러한 요구 사항을 충족하기 위해 어떤 서비스 세트를 권장해야 합니까?

## 문제 130

애플리케이션이 여러 가용 영역에 걸쳐 Amazon EC2 인스턴스에서 실행됩니다. 이 인스턴스들은 Application Load Balancer 뒤의 Amazon EC2 Auto Scaling 그룹에서 실행됩니다. 애플리케이션은 EC2 인스턴스의 CPU 사용률이 40%이거나 이에 근접할 때 가장 잘 작동합니다. 그룹 내의 모든 인스턴스에서 원하는 성능을 유지하려면 솔루션 아키텍트는 어떻게 해야 합니까?

- **A.** 단순 조정(simple scaling) 정책을 사용하여 Auto Scaling 그룹을 동적으로 조정합니다.
    
- **B.** 대상 추적(target tracking) 정책을 사용하여 Auto Scaling 그룹을 동적으로 조정합니다.
    
- **C.** AWS Lambda 함수를 사용하여 Auto Scaling 그룹의 원하는 용량을 업데이트합니다.
    
- **D.** 예약된 조정(scheduled scaling) 작업을 사용하여 Auto Scaling 그룹을 스케일 업 및 스케일 다운합니다

>[!.questions]- 풀이
>정답 : B
>EC2 Auto Scaling 조정 정책
>단순 조정 정책 : "CPU 50%가 넘으면 인스턴스 2개 추가"와 같이 단계별 규칙을 정해야됨. 정교한 작업에는 부적절
>대상 추적 정책 : "평균 CPU 40%를 유지해줘"하면 AWS가 알아서 인스턴스를 늘리거나 줄이면서 유지한다.
>예약된 조정 정책 : 시간을 기준으로 인스턴스를 조절한다.
