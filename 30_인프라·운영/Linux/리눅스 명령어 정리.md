
## 파일 시스템 & 기본 탐색

```bash
# 현재 위치 출력
pwd 

# 파일 목록
ls -la (= ll)

# 디렉토리 이동
cd /경로

# 디렉토리 생성(하위까지 한번에)
mkdir -p

# 강제 삭제
rm -rf

# 복사
cp

# 이동
mv

# 파일 검색
find / -name "파일명"


```

## 파일 내용 보기

```bash
# 파일 출력
cat / less / more 

# 로그 실시간 모니터링
tail -f /var/log/...

# 내용 검색
grep "키워드" 파일

# 재귀 검색
grep -r "키워드" ./
```

## 권한 관리

```bash
# 소유자만 읽기/쓰기, 나머진 읽기만
chmod 644 

# 실행은 모두, 수정은 소유자만
chmod 755

# 소유자만 
chmod 600

# 권한 모두에게
chmod 777


```

## 프로세스 관리

```bash
# 실행 중인 프로세스 전체 조회
ps aux

# 실시간 CPU/메모리 모니터링
top /htop

# 프로세스 강제 종료
kill -9 PID

# 서비스 관리
systemctl start/stop/status 서비스명
```

## 네트워크 

```bash
# 연결 확인, HTTP 요청
ping / curl / wget 

# 열린 포트 확인
ss -tuln

# ip 주소 확인
ip addr / ifconfig

# 원격 서버 접속
ssh user@host

# 원격으로 파일 복사
scp 파일 user@host:/경로

```

## 디스크 / 메모리

```bash
# 디스크 사용량
df -h

# 특정 디렉토리 용량
du -sh ./

# 메모리 사용량
free -h
```
