비밀 관리 Parameter Store SecureString + IAM Role 조회, 코드·이미지에 비밀 X

📍 GPS 값 유실 방지

App → Rule 컨테이너 경로를 SQS로 분리. Rule 처리가 죽어도 GPS 이벤트는 큐에 살아남는다.

App ↔ Rule 분리 App은 SQS publish 후 즉시 응답, Rule 컨테이너가 비동기 수신
가시성 Timeout Rule 컨테이너 처리 중 죽어도 메시지 큐 복귀 → 유실 0
DLQ N회 실패 메시지 격리, 무한 재시도 차단 · 사후 분석 가능
폭주 흡수 GPS 트래픽 급증에도 Rule 컨테이너 평탄 처리 (백프레셔)
분산 영속화 SQS 자체가 다중 AZ 저장 → 컨테이너 다 죽어도 메시지 생존
App → Rule 경로의 어느 컴포넌트가 죽어도 GPS 이벤트 유실 0.
